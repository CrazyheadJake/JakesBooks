# Personal Book List

A full-stack web application to track reading progress, manage a personal library, and rate books. Built with the MERN stack (MongoDB, Express, React, Node.js).

## Features

* **User Authentication:** Secure login/signup with session-based authentication.
* **Library Management:** Add, edit, and delete book entries.
* **Progress Tracking:** Track reading dates, ratings (0-100), and series information.
* **Search & Sort:** Filter books by genre or sort by rating, date, or author.
* **Responsive Design:** Mobile-friendly interface built with Bootstrap.
* **Review System:** Write and store personal reviews for each book.

## Tech Stack

**Client:**
* React (Vite)
* Bootstrap 4/5 (UI Framework)
* TypeScript

**Server:**
* Node.js & Express
* MongoDB (Database)
* Express-Session (Authentication)
* MongoDB Native Driver
